Smart Agriculture System

Automated farming using Arduino and sensors

🌱 Project Overview

This project automates agricultural tasks using sensors and motors.

  • ☔ **Rain Sensor** - Detects rain and moves the servo.
  • 🌿 **Soil Moisture Sensor** - Checks soil dryness.
  • 🌡️ **Humidity Sensor** - Monitors environmental conditions.
  • 🔄 **Servo Motor** - Opens/Closes a cover based on rain detection.

📜 Arduino Code

#include <Servo.h>
Servo myServo;
int rainSensor = A0;
void setup() {
    myServo.attach(9);
    pinMode(rainSensor, INPUT);
}
void loop() {
    int rainValue = analogRead(rainSensor);
    if (rainValue < 500) { 
        myServo.write(90);
    } else {
        myServo.write(0);
    }
    delay(1000);
}
