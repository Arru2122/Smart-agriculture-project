Smart Agriculture System

Automated farming using Arduino and sensors

🌱 Project Overview

This project automates agricultural tasks using various sensors and motors.

🔧 Components Used

☔ Rain Sensor

Detects rainfall and triggers the servo motor to cover crops.

🌿 Soil Moisture Sensor

Measures the moisture level in the soil and triggers irrigation.

🌡️ Humidity Sensor

Monitors environmental humidity to analyze crop health.

🔄 Servo Motor

Moves 90° when rain is detected to cover plants.

⚡ Arduino Uno

Controls all sensors and actuators in the system.

📜 Arduino Code

#include <Servo.h>
Servo myServo;
int rainSensor = A0;
void setup() {
    myServo.attach(9);
    pinMode(rainSensor, INPUT);
}
void loop() {
    int rainValue = analogRead(rainSensor);
    if (rainValue < 500) { 
        myServo.write(90);
    } else {
        myServo.write(0);
    }
    delay(1000);
}
